# Popup UI

## 역할
- 브라우저 툴바에서 열리는 확장 프로그램 팝업 뷰를 담당합니다.
- 백그라운드 서비스 워커와의 기본 연결 상태(PING)를 확인합니다.

## 구성 파일
| 경로 | 설명 |
| --- | --- |
| `index.html` | 팝업 HTML 컨테이너. `#root` 엘리먼트와 `main.tsx` 스크립트 엔트리로 구성됩니다. |
| `main.tsx` | React 부트스트랩 코드. 전역 스타일과 `Popup` 컴포넌트를 로드합니다. |
| `Popup.tsx` | 팝업 뷰 컴포넌트. 백그라운드 메시지 전송과 상태 표시를 구현합니다. |

## 런타임 동작
1. 컴포넌트 마운트 시 `sendMessage({ type: 'PING' })`를 호출해 서비스 워커와의 통신을 확인합니다.
2. 요청 실패 시 상태를 `error`로 전환하고 사용자에게 로그 확인을 안내합니다.
3. 기본 UI는 전역 스타일(`panel`, `subtitle`, `hint`) 클래스를 재사용합니다.

## 확장 포인트
- 상태를 늘리고 싶다면 `status` 스테이트를 enum으로 확장하고 메시지 페이로드/응답 타입을 `shared/types.ts`에 추가하세요.
- 팝업에서 설정을 제어하려면 `shared/storage`를 사용해 Chrome 스토리지와 연동할 수 있습니다.
