1. 이 프로젝트는 사용자가 개인정보를 적절하지 못한 곳에 필터링 없이 입력하는 걸 방지하기 위한 개인정보 보호 익스텐션입니다.
2. 사용자가 페이지에 접속하면, 사전 정의, 주입된 FormFilter의 조건에 따라 필터링된 일부 입력 폼/필드에 shadow dom을 활용하여 미러 폼을 만듭니다.
3. 미러 폼 내부, 오른쪽 끝 정도에 클릭 가능한 아이콘 버튼을 넣습니다. 사용자가 입력하면 아이콘 버튼이 나타나고, 입력된 텍스트가 없으면 토글 버튼이 사라집니다.
4. 사용자가 텍스트를 입력하면서 실시간 검사를 수행하는데, 검사 전략은 마찬가지로 사전 정의된 전략(정규식, API 등) 중 하나를 주입받아 적용합니다.
5. 실시간 검사 결과 개인정보라고 판단되어 마스킹이 필요한 곳에는 밑줄 혹은 하이라이트 처리를 합니다.
6. 밑줄 혹은 하이라이트 처리가 된 곳은 클릭 및 호버 가능하도록 하여 검사 결과의 내용과 마스킹 처리를 할건지에 대한 버튼을 팝업으로 띄워줍니다.
7. 3에서 넣은 아이콘 버튼을 누르면 해당 폼에 있는 전체 텍스트를 검사합니다. 검사 결과와 마스킹 처리 버튼을 보여주는 것은 마찬가지로 팝업으로 처리합니다.
8. 텍스트를 선택하고 오른쪽 마우스를 누르면 다른 크롬 익스텐션처럼 텍스트 검사하기 기능을 실행할 수 있도록 목록에 보여줘야 합니다.
9. 사용자가 마스킹 처리 버튼을 누르면, 해당 텍스트를 별표로 마스킹 처리합니다.
